# ContactApp
## Project Description
Simpel Contact App for Learning CRUD

### How to Install and Run the Project
1.  Install Node
2.  Clone this repository 
    ```
    git clone https://github.com/alitdarmaputra/ContactApp.git
    ```
3. Enter the Directory `ContactApp` 
    ```
    cd ContactApp
    ```
5. Run the program 
    ```
    npm start
    ```
